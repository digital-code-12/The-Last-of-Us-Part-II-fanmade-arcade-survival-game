<h1>Additional requirements to run the game</h1>
<p>Bellow is a list of the requirements to run the game.</p>
<h2>Game requirements:</h2>
<p>For the game to run, a valid installation of python needs to be installed. You can install it here: https://www.python.org/downloads/</p>
<p>In addition, after the installation is complete, open the command prompt (cmd) and type in: </p>
<code>pip install pygame</code>
<p></p>
<p>This command will install the python module pygame on your computer.</p>
